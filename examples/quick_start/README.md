# Phylon Quick Start

Welcome to the Quick Start example for Phylon. This folder demonstrates how to configure and run a deterministic artificial life simulation using Rhai scripting and RON configuration files.

## Files

- `simulation.rhai`: The entry point script that initializes the ecosystem.
- `config.ron`: Global physics and mutation settings.
- `overlays.ron`: Initial rendering states for diagnostic overlays.
- `species.ron`: Definitions of initial organism behavior archetypes.
- `camera.ron`: Viewport configuration.
- `seed_42_snapshot.bin`: A pre-simulated backup snapshot (placeholder).

## How to Run
Load this directory via the **File -> Open Experiment** menu in the Phylon UI, or execute the script directly using **Run -> Run Script...**
